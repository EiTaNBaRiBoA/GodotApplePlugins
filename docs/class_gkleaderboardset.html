GKLeaderboardSet¶

Inherits: RefCounted < Object

Groups related Apple Game Center leaderboards.

Description¶

GKLeaderboardSet mirrors Apple’s “leaderboard set” concept. Although the current binding only exposes the opaque object, you can keep references to sets returned by other APIs to organise UI the same way as on iOS. See Apple’s GKLeaderboardSet reference for the native API guide.
